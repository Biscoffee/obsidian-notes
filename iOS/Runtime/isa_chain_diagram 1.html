ObjC Runtime — isa & superclass 走位图
Root class 在顶部，superclass 向上指，nil 在最顶端
isa 链（水平）：实例 → 类对象 → 元类，所有元类 isa 通过右侧弧线收敛到根元类，根元类 isa 指向自身（loop）。   superclass 链（垂直向上）：类列 Subclass → Superclass → Root class → nil；元类列平行向上。   关键特例：Root class (meta) 的 superclass 跨列指向 Root class（类对象，不是元类），使消息在元类链顶端能回退到 NSObject 实例方法继续查找。
