ObjC Runtime — isa & superclass 走位图
apple-oss-distributions / objc4 · 三层继承：MySubclass → MyClass → NSObject
isa 链：实例 → 类对象 → 元类 → 根元类（NSObject meta）→ 自身（loop）。无论继承多深，所有元类的 isa 都跳过中间层，直接收敛到根元类。   superclass 链（类列）：正常继承，MySubclass → MyClass → NSObject → nil。   superclass 链（元类列）：平行于类列，但根部有一条特殊跨列箭头——NSObject meta 的 superclass 指向 NSObject 类对象（不是元类！），这使得向类发送消息找不到类方法时，能回落到 NSObject 的实例方法上继续查找。
